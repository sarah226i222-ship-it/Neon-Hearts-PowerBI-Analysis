# Neon-Hearts-PowerBI-Analysis
Analysis of a dating app marketing campaign using Power BI
The Challenge: To evaluate campaign effectiveness by identifying user behavior patterns and high-value segments.

Technical Skills Applied:

Data Transformation: Cleaned and prepared the dataset using Power BI to ensure formatting accuracy.

KPI Dashboarding: Designed interactive cards to track core metrics: Total Matches (113) and Dates Planned (19).

Trend Analysis: Visualized user activity over time, identifying significant "Weekend Spikes" in engagement.

Advanced Aggregation: Utilized Distinct Count on User IDs to accurately measure unique premium subscribers, avoiding data duplication.

Interactive Filtering: Implemented Slicers for demographic deep-dives by gender and age group.

Key Insights:

The analysis revealed that female users had a higher success rate in planning dates during the campaign.

Weekend activity is the primary driver for matches, suggesting optimized times for push notifications.
